# 難易度レベル定義（Cambridge Dictionary 準拠）

4択・ディクテーションとも、表示と問題の難易度は **Cambridge Dictionary / English Profile (CEFR)** のレベル定義に合わせています。

- 表記: [Cambridge Dictionary Help - A1, B2, C1 labels](https://dictionary.cambridge.org/help/)
- 語彙の目安: English Vocabulary Profile (EVP) の A1〜C2 が参考になる

## アプリ内の対応

| CEFR | Cambridge ラベル      | TOEIC 帯 | 問題レベル (Firestore) |
|------|----------------------|----------|-------------------------|
| A2   | Elementary           | 400      | level 1–2               |
| B1   | Intermediate         | 600      | level 3–4                |
| B2   | Upper-Intermediate   | 730      | level 5–6                |
| C1   | Advanced             | 860      | level 7–8                |
| C2   | Proficiency          | 990      | level 9–10               |

※ A1 Beginner は現状の TOEIC 帯では未対応（必要なら 400 未満の帯を追加する想定）。

## 問題バンクでの運用

- **level 1–2**: A2 相当の語彙・文法（基本前置詞、三単現、簡単な依頼など）
- **level 3–4**: B1 相当（受動態、ビジネス基礎、依頼・説明）
- **level 5–6**: B2 相当
- **level 7–8**: C1 相当
- **level 9–10**: C2 相当

語彙を追加・見直す場合は、[English Profile](https://www.englishprofile.org/) や Cambridge の試験別語彙リスト（例: A2 Key）を参照すると整合しやすい。
